# ChatGPT로 엑셀 수식 만드는 법 — 복잡한 함수도 3초 만에 해결

엑셀에서 VLOOKUP 수식을 만들다가 에러가 나서 한 시간째 검색하고 계신가요? 이제 ChatGPT에게 한국어로 "A열에서 이름을 찾아서 B열의 매출을 가져와줘"라고 말하면 됩니다. 복잡한 함수 문법을 외울 필요가 없습니다.

> 핵심 답변: ChatGPT에게 "어떤 데이터에서, 어떤 조건으로, 어떤 결과를 원하는지" 자연어로 설명하면 정확한 엑셀 수식을 즉시 받을 수 있습니다.

이 글에서는 실무에서 가장 많이 쓰는 엑셀 함수 10가지에 대해, ChatGPT에게 정확히 어떻게 물어봐야 하는지 프롬프트 템플릿을 제공합니다. 복사해서 바로 사용하세요.

---

## ChatGPT에게 엑셀 수식을 요청하는 기본 원칙

ChatGPT는 엑셀 수식 생성에 매우 능숙합니다. 하지만 질문하는 방식에 따라 결과의 정확도가 크게 달라집니다. 다음 원칙을 기억하세요.

### 좋은 프롬프트 vs 나쁜 프롬프트

| 구분 | 나쁜 프롬프트 | 좋은 프롬프트 |
|------|-------------|-------------|
| 예시 1 | "VLOOKUP 써줘" | "A열에 제품명, B열에 가격이 있는 표에서, D1에 입력한 제품명의 가격을 찾는 VLOOKUP 수식을 만들어줘" |
| 예시 2 | "조건부 합계 수식" | "C열이 '서울'이고 D열이 2026년 3월인 행들의 E열 매출 합계를 구하는 SUMIFS 수식" |
| 예시 3 | "날짜 계산해줘" | "A1 셀의 입사일로부터 오늘까지의 근무 연수를 구하는 수식 (소수점 1자리)" |

### 프롬프트 작성 3단계 공식

효과적인 엑셀 수식 프롬프트는 3가지 요소를 포함해야 합니다.

**1단계 - 데이터 구조 설명**
```
"A열: 직원명, B열: 부서, C열: 직급, D열: 월급여, E열: 입사일이 있는 표가 있어."
```

**2단계 - 원하는 결과 설명**
```
"부서가 '영업팀'인 직원들의 평균 월급여를 구하고 싶어."
```

**3단계 - 추가 조건 명시**
```
"결과는 천 단위 구분 쉼표가 있는 형태로 보여주고, 해당 부서가 없으면 '해당없음'이라고 표시해줘."
```

이 세 가지를 하나로 합치면 ChatGPT가 정확한 수식을 만들어 줍니다. 아래에서 함수별로 실제 프롬프트 예시를 보겠습니다.

## 함수별 프롬프트 예시 10개 — 복사해서 바로 사용

### 1. VLOOKUP / XLOOKUP — 데이터 찾기

VLOOKUP은 엑셀에서 가장 많이 검색되는 함수입니다. 최신 버전이라면 XLOOKUP을 쓰는 것이 더 유연합니다.

**이렇게 물어보세요 (VLOOKUP)**
```
엑셀 수식 만들어줘.
- Sheet1의 A열에 제품코드, B열에 제품명, C열에 단가가 있어.
- Sheet2의 A2 셀에 제품코드를 입력하면
- Sheet1에서 해당 제품의 단가를 찾아서 Sheet2의 B2에 표시하고 싶어.
- 제품코드가 없으면 "미등록"이라고 표시해줘.
```

**ChatGPT가 만들어주는 수식**
```
=IFERROR(VLOOKUP(A2,Sheet1!A:C,3,FALSE),"미등록")
```

**이렇게 물어보세요 (XLOOKUP)**
```
XLOOKUP으로 같은 기능을 만들어줘.
- Sheet1의 A열에서 제품코드를 찾고
- C열의 단가를 반환
- 못 찾으면 "미등록" 표시
```

**ChatGPT가 만들어주는 수식**
```
=XLOOKUP(A2,Sheet1!A:A,Sheet1!C:C,"미등록")
```

**결과**: VLOOKUP은 왼쪽에서 오른쪽으로만 검색하지만, XLOOKUP은 방향 제한이 없어서 더 유연합니다. ChatGPT에게 두 버전 모두 요청하면 자신의 엑셀 버전에 맞는 것을 선택할 수 있습니다.

> 팁: "내 엑셀이 2019 버전이야"라고 추가하면 ChatGPT가 해당 버전에서 지원하는 함수만 사용합니다.

---

### 2. IF / IFS — 조건 판단

IF 함수는 조건에 따라 다른 결과를 표시할 때 사용합니다. 조건이 여러 개일 때는 IFS가 편합니다.

**이렇게 물어보세요 (중첩 IF)**
```
엑셀 수식 만들어줘.
- D열에 점수가 있어 (0~100).
- 90 이상이면 "A", 80 이상이면 "B", 70 이상이면 "C",
  60 이상이면 "D", 그 미만이면 "F"를 E열에 표시하고 싶어.
- D2 셀부터 시작해.
```

**ChatGPT가 만들어주는 수식**
```
=IF(D2>=90,"A",IF(D2>=80,"B",IF(D2>=70,"C",IF(D2>=60,"D","F"))))
```

**이렇게 물어보세요 (IFS)**
```
위와 같은 조건인데 IFS 함수로 만들어줘. 더 깔끔하게.
```

**ChatGPT가 만들어주는 수식**
```
=IFS(D2>=90,"A",D2>=80,"B",D2>=70,"C",D2>=60,"D",TRUE,"F")
```

**결과**: 중첩 IF는 괄호가 많아서 읽기 어렵지만, IFS는 조건-결과 쌍으로 나열하므로 직관적입니다. 다만 IFS는 Excel 2019 이상에서만 사용 가능합니다.

---

### 3. SUMIFS / COUNTIFS — 조건부 합계와 개수

여러 조건을 동시에 만족하는 데이터의 합계나 개수를 구할 때 사용합니다.

**이렇게 물어보세요 (SUMIFS)**
```
엑셀 수식 만들어줘.
- A열: 날짜, B열: 지역, C열: 제품, D열: 매출액
- 조건: 지역이 "서울"이고, 날짜가 2026년 3월인 행들의 매출액 합계
- 날짜는 yyyy-mm-dd 형식이야.
```

**ChatGPT가 만들어주는 수식**
```
=SUMIFS(D:D,B:B,"서울",A:A,">="&DATE(2026,3,1),A:A,"<"&DATE(2026,4,1))
```

**이렇게 물어보세요 (COUNTIFS)**
```
같은 조건에서 해당하는 행의 개수를 구해줘.
```

**ChatGPT가 만들어주는 수식**
```
=COUNTIFS(B:B,"서울",A:A,">="&DATE(2026,3,1),A:A,"<"&DATE(2026,4,1))
```

**결과**: SUMIFS와 COUNTIFS의 날짜 조건은 실수가 잦은 부분입니다. ChatGPT는 DATE 함수를 활용한 정확한 날짜 범위를 자동으로 설정해 줍니다.

> 팁: "조건이 3개 이상일 때도 되나요?"라고 추가 질문하면 ChatGPT가 확장된 수식을 제공합니다.

### 4. 피벗 테이블 관련 — 데이터 요약

피벗 테이블은 수식은 아니지만, ChatGPT에게 만드는 방법을 단계별로 물어볼 수 있습니다.

**이렇게 물어보세요**
```
엑셀에서 피벗 테이블 만드는 방법을 알려줘.
- 데이터: A열(날짜), B열(부서), C열(항목), D열(금액)
- 원하는 결과: 부서별, 월별 금액 합계를 한눈에 보고 싶어.
- 행에 부서, 열에 월, 값에 금액 합계를 넣고 싶어.
- 단계별로 설명해줘.
```

**ChatGPT 답변 요약**
1. 데이터 범위를 선택합니다 (A1부터 데이터 끝까지).
2. [삽입] 탭 > [피벗 테이블]을 클릭합니다.
3. 새 워크시트에 만들기를 선택하고 확인을 누릅니다.
4. 오른쪽 필드 목록에서:
   - "부서"를 행 영역으로 드래그합니다.
   - "날짜"를 열 영역으로 드래그합니다.
   - "금액"을 값 영역으로 드래그합니다.
5. 열 영역의 "날짜"를 우클릭하고 [그룹화]에서 "월"을 선택합니다.

**결과**: ChatGPT는 피벗 테이블의 설정 단계를 한국어 엑셀 메뉴 기준으로 안내합니다. "영문 엑셀 기준으로 알려줘"라고 하면 영문 메뉴명으로 전환됩니다.

> 팁: 피벗 테이블 결과를 수식으로 구현하고 싶다면 "피벗 테이블 대신 SUMIFS로 같은 결과를 만들어줘"라고 요청하세요.

### 5. 날짜 함수 — 근무일수, 나이, 기간 계산

날짜 관련 수식은 직장인이 가장 자주 필요로 하면서도 가장 헷갈려하는 영역입니다.

**이렇게 물어보세요 (근무연수)**
```
엑셀 수식 만들어줘.
- A열에 입사일이 yyyy-mm-dd 형식으로 있어.
- 오늘 날짜 기준으로 근무 연수를 계산하고 싶어.
- "3년 4개월" 형태의 텍스트로 표시해줘.
```

**ChatGPT가 만들어주는 수식**
```
=DATEDIF(A2,TODAY(),"Y")&"년 "&DATEDIF(A2,TODAY(),"YM")&"개월"
```

**이렇게 물어보세요 (영업일 계산)**
```
A1(시작일)과 B1(종료일) 사이의 영업일 수를 구해줘.
주말(토, 일)은 제외하고, 공휴일 목록이 Sheet2의 A열에 있어.
```

**ChatGPT가 만들어주는 수식**
```
=NETWORKDAYS(A1,B1,Sheet2!A:A)
```

**이렇게 물어보세요 (만 나이)**
```
A열에 생년월일이 있어.
오늘 기준 만 나이를 구하는 수식 만들어줘.
```

**ChatGPT가 만들어주는 수식**
```
=DATEDIF(A2,TODAY(),"Y")
```

**결과**: DATEDIF는 엑셀에서 공식적으로 문서화되지 않은 함수이지만 여전히 작동합니다. ChatGPT는 이런 숨겨진 함수도 잘 알고 있어서 날짜 계산에 매우 유용합니다.

### 6. 텍스트 함수 — 문자열 처리

이메일 주소에서 도메인 추출, 이름과 성 분리, 전화번호 형식 변경 등 텍스트 처리 작업에 유용합니다.

**이렇게 물어보세요 (이메일 도메인 추출)**
```
A열에 이메일 주소가 있어 (예: hong@naver.com).
@ 뒤의 도메인 부분만 추출하는 수식 만들어줘.
```

**ChatGPT가 만들어주는 수식**
```
=MID(A2,FIND("@",A2)+1,LEN(A2)-FIND("@",A2))
```

**이렇게 물어보세요 (전화번호 포맷)**
```
A열에 "01012345678" 형태의 전화번호가 있어.
"010-1234-5678" 형태로 변환하는 수식 만들어줘.
```

**ChatGPT가 만들어주는 수식**
```
=LEFT(A2,3)&"-"&MID(A2,4,4)&"-"&RIGHT(A2,4)
```

**이렇게 물어보세요 (성명 분리)**
```
A열에 "홍길동" 같은 한국 이름이 있어.
첫 글자(성)와 나머지(이름)를 각각 B열, C열에 분리하는 수식.
```

**ChatGPT가 만들어주는 수식**
```
B열(성): =LEFT(A2,1)
C열(이름): =MID(A2,2,LEN(A2)-1)
```

**결과**: 텍스트 함수는 조합이 복잡해질수록 ChatGPT의 가치가 높아집니다. "두 글자 성(남궁, 선우 등)도 처리하고 싶어"라고 추가하면 더 정교한 수식을 받을 수 있습니다.

### 7. INDEX-MATCH — 고급 데이터 검색

VLOOKUP의 한계를 넘어서는 강력한 조합입니다. 왼쪽 방향 검색, 다중 조건 검색이 가능합니다.

**이렇게 물어보세요 (기본)**
```
엑셀 수식 만들어줘. INDEX-MATCH 조합으로.
- A열: 직원번호, B열: 이름, C열: 부서, D열: 연봉
- F1에 이름을 입력하면 해당 직원의 직원번호를 G1에 표시하고 싶어.
  (왼쪽 방향 검색이라 VLOOKUP으로는 안 돼.)
```

**ChatGPT가 만들어주는 수식**
```
=INDEX(A:A,MATCH(F1,B:B,0))
```

**이렇게 물어보세요 (다중 조건)**
```
A열: 부서, B열: 직급, C열: 이름, D열: 연봉
"영업팀" 부서의 "과장" 직급에 해당하는 연봉을 찾는 수식.
INDEX-MATCH로 다중 조건 검색해줘.
```

**ChatGPT가 만들어주는 수식**
```
=INDEX(D:D,MATCH(1,(A:A="영업팀")*(B:B="과장"),0))
```
(이 수식은 Ctrl+Shift+Enter로 배열 수식으로 입력해야 합니다. Excel 365에서는 그냥 Enter로 됩니다.)

**결과**: INDEX-MATCH는 처음 보면 어렵지만, ChatGPT에게 상황을 설명하면 자동으로 최적의 조합을 만들어 줍니다. VLOOKUP으로 해결이 안 될 때 "INDEX-MATCH로 바꿔줘"라고 요청하면 됩니다.

### 8. 조건부 서식 — 시각적 강조

조건부 서식은 수식이 아니라 서식 규칙이지만, 수식 기반 조건부 서식은 ChatGPT가 잘 만들어줍니다.

**이렇게 물어보세요**
```
엑셀 조건부 서식용 수식 만들어줘.
- D열에 매출액이 있어.
- 각 행의 매출액이 해당 열 평균보다 높으면 초록색,
  평균보다 낮으면 빨간색으로 배경을 칠하고 싶어.
- D2 셀부터 적용할 수식을 알려줘.
```

**ChatGPT가 만들어주는 수식**
```
초록색 조건: =D2>AVERAGE(D:D)
빨간색 조건: =D2<AVERAGE(D:D)
```

**적용 방법** (ChatGPT가 함께 안내)
1. D열 전체를 선택합니다.
2. [홈] > [조건부 서식] > [새 규칙]을 클릭합니다.
3. "수식을 사용하여 서식을 지정할 셀 결정"을 선택합니다.
4. 위 수식을 입력하고 원하는 서식(배경색)을 지정합니다.

**이렇게 물어보세요 (고급)**
```
마감일(E열)이 오늘 기준 3일 이내면 노란색,
이미 지난 경우 빨간색으로 표시하는 조건부 서식 수식.
```

**ChatGPT가 만들어주는 수식**
```
빨간색: =AND(E2<TODAY(),E2<>"")
노란색: =AND(E2>=TODAY(),E2<=TODAY()+3,E2<>"")
```

**결과**: 조건부 서식은 설정 과정이 복잡한데, ChatGPT에게 수식과 함께 적용 방법까지 물어보면 단계별로 안내받을 수 있습니다.

### 9. 매크로 / VBA — 반복 작업 자동화

VBA 코드는 문법이 완전히 다르기 때문에 ChatGPT의 도움이 가장 빛나는 영역입니다.

**이렇게 물어보세요**
```
엑셀 VBA 매크로 만들어줘.
- 현재 시트의 A열에 파일 경로 목록이 있어.
- 각 경로의 파일이 실제로 존재하는지 확인해서
- B열에 "존재" 또는 "없음"을 표시하는 매크로.
- A2부터 데이터가 비어있는 행까지 실행.
```

**ChatGPT가 만들어주는 VBA 코드**
```vba
Sub CheckFileExists()
    Dim ws As Worksheet
    Dim lastRow As Long
    Dim i As Long
    Dim filePath As String

    Set ws = ActiveSheet
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row

    For i = 2 To lastRow
        filePath = ws.Cells(i, 1).Value
        If Dir(filePath) <> "" Then
            ws.Cells(i, 2).Value = "존재"
        Else
            ws.Cells(i, 2).Value = "없음"
        End If
    Next i

    MsgBox "파일 확인 완료!", vbInformation
End Sub
```

**이렇게 물어보세요 (자동 보고서)**
```
매주 월요일마다 실행되는 VBA 매크로 만들어줘.
- "원본데이터" 시트에서 이번 주 데이터만 필터링
- "주간보고" 시트에 부서별 합계를 정리
- 결과를 PDF로 저장 (파일명: 주간보고_2026-03-31.pdf)
```

**결과**: VBA는 직접 작성하면 디버깅에 많은 시간이 걸리지만, ChatGPT는 즉시 실행 가능한 코드를 제공합니다. "에러가 나요"라고 에러 메시지를 붙여넣으면 수정 코드도 바로 받을 수 있습니다.

> 주의: VBA 매크로가 포함된 파일은 .xlsm 형식으로 저장해야 합니다. .xlsx로 저장하면 매크로가 사라집니다.

### 10. 차트 만들기 — 데이터 시각화

차트는 수식이 아니지만, ChatGPT에게 데이터에 맞는 최적의 차트 유형과 설정 방법을 물어볼 수 있습니다.

**이렇게 물어보세요**
```
엑셀 차트 추천해줘.
- A열: 월(1월~12월), B열: 매출, C열: 비용, D열: 이익
```

**이렇게 물어보세요 (차트 VBA)**

```
색상은 매출: 파랑, 비용: 빨강, 이익률: 초록 점선으로 설정.
```

**결과**: 차트 유형 선택이 어려울 때 ChatGPT에게 데이터의 성격과 목적을 설명하면 최적의 차트를 추천받을 수 있습니다.

## 엑셀 데이터를 ChatGPT에 붙여넣기하는 팁

ChatGPT에게 수식을 요청할 때, 실제 데이터를 보여주면 훨씬 정확한 결과를 받을 수 있습니다. 올바른 방법을 알아보겠습니다.

### 방법 1: 표 형태로 직접 붙여넣기

엑셀에서 셀 범위를 복사(Ctrl+C)한 후 ChatGPT 채팅창에 붙여넣기(Ctrl+V)하면 탭으로 구분된 텍스트가 입력됩니다.

```
이름    부서    매출
홍길동  영업팀  5000000
김영희  마케팅  3200000
이철수  영업팀  4800000
```

ChatGPT는 이 형태를 잘 인식합니다. 데이터와 함께 질문하세요.

```
위 데이터에서 영업팀의 매출 합계를 구하는 SUMIF 수식을 만들어줘.
실제 셀 위치 기준으로 (A1이 "이름" 헤더).
```

### 방법 2: 스크린샷 활용

ChatGPT Plus(유료)를 사용한다면 엑셀 화면을 캡처해서 이미지로 올릴 수 있습니다.

- 스크린샷을 찍을 때 열 머리글(A, B, C...)과 행 번호(1, 2, 3...)가 보이도록 합니다.
- "이 엑셀 화면을 보고 C열의 합계를 구하는 수식을 만들어줘"라고 요청합니다.

### 방법 3: 구조만 설명하기

데이터가 많거나 민감할 때는 구조만 설명합니다.

```
A1:A100에 제품명(텍스트), B1:B100에 수량(정수),
C1:C100에 단가(원 단위), D열에 금액(수량x단가) 수식을 넣고 싶어.
마지막 행 아래에 합계도 자동으로 구해줘.
```

### 데이터 붙여넣기 시 주의사항

| 상황 | 주의사항 | 해결법 |
|------|---------|--------|
| 데이터가 많을 때 | 전체 데이터를 붙여넣을 필요 없음 | 첫 5~10행만 샘플로 제공 |
| 개인정보 포함 | 이름, 전화번호 등 민감정보 주의 | 더미 데이터로 대체 후 요청 |
| 특수 형식 | 날짜, 통화 등 서식이 깨질 수 있음 | 형식을 텍스트로 명시 |
| 수식 결과값 | 수식이 아닌 결과값이 복사됨 | "이 셀에는 수식이 들어있어"라고 명시 |

## Google Sheets에서도 동일하게 쓰는 법

ChatGPT가 만들어준 엑셀 수식은 대부분 Google Sheets에서도 그대로 작동합니다. 다만 일부 차이점이 있습니다.

### 엑셀 vs Google Sheets 함수 차이

| 기능 | 엑셀 | Google Sheets | 비고 |
|------|------|--------------|------|
| 찾기 | VLOOKUP / XLOOKUP | VLOOKUP / XLOOKUP | 동일 |
| 조건부 합계 | SUMIFS | SUMIFS | 동일 |

| 이미지 | 없음 | IMAGE | Sheets 전용 |
| 매크로 | VBA | Apps Script (JavaScript) | 완전히 다름 |

### Google Sheets 전용 프롬프트

```
Google Sheets 수식 만들어줘 (엑셀 아님).
- A열에 영어 텍스트가 있어.
- B열에 한국어 번역을 자동으로 넣고 싶어.
```

**ChatGPT가 만들어주는 수식**

```
=GOOGLETRANSLATE(A2,"en","ko")
```

Google Sheets에서 매크로가 필요할 때는 VBA 대신 Apps Script를 요청합니다.

```

- 매일 오전 9시에 자동 실행
- A열의 URL 목록에서 각 웹페이지의 제목(title)을 가져와서
- B열에 입력하는 스크립트.
```

> 팁: ChatGPT에게 "이 엑셀 수식을 Google Sheets 버전으로 변환해줘"라고 요청하면 호환되지 않는 부분을 자동으로 수정해 줍니다.

## 자주 발생하는 에러와 ChatGPT로 해결하기

엑셀 수식에서 에러가 발생했을 때 ChatGPT에게 해결을 요청하는 방법입니다.

### 에러 유형별 프롬프트

| 에러 | 의미 | ChatGPT에게 물어보는 법 |
|------|------|----------------------|
| #N/A | 값을 찾을 수 없음 | "VLOOKUP에서 #N/A 에러가 나. 수식은 =VLOOKUP(A2,Sheet2!A:C,3,FALSE)이고 A2값은 'ABC-001'인데 Sheet2에는 분명 있어" |
| #REF! | 잘못된 셀 참조 | "=INDEX(A:A,MATCH(F1,B:B,0))에서 #REF! 에러. F1에 '홍길동' 입력했어" |
| #VALUE! | 잘못된 데이터 유형 | "=A2+B2에서 #VALUE! 에러. A2는 숫자인데 B2가 텍스트 형식인 것 같아" |
| #DIV/0! | 0으로 나눔 | "=A2/B2에서 B2가 0일 때 에러 안 나게 해줘" |
| #NAME? | 함수명 오류 | "=XLOOKUP이 #NAME? 에러. 내 엑셀에서 지원 안 하는 건가?" |
| 순환참조 | 수식이 자기 자신을 참조 | "순환참조 경고가 뜨는데, 수식은 =SUM(A1:A10)이고 이 수식이 A5에 있어" |

### 에러 해결 프롬프트 템플릿

다음 템플릿을 복사해서 사용하세요.

```
엑셀 수식 에러를 해결해줘.
- 수식: [에러가 나는 수식을 붙여넣기]
- 에러: [#N/A, #REF! 등]
- 셀 위치: [수식이 있는 셀]
- 데이터: [관련 데이터 샘플 5행 정도]
- 의도: [원래 하려던 것]
- 엑셀 버전: [2019, 365 등]
```

## ChatGPT 엑셀 활용 시 주의사항

### 대용량 데이터 처리

- ChatGPT에게 데이터를 보낼 때는 전체 데이터가 아닌 **샘플(5~10행)**만 보내세요.
- 수만 행의 데이터를 전부 붙여넣으면 ChatGPT가 제대로 처리하지 못합니다.
- "데이터가 5만 행이야"라고 행 수를 알려주면 성능을 고려한 수식을 만들어 줍니다.

### 개인정보 보호

| 항목 | 위험도 | 대응 방법 |
|------|--------|----------|
| 직원 이름 | 중간 | 더미 이름으로 대체 (홍길동, 김철수 등) |
| 전화번호 | 높음 | 임의 번호로 대체 또는 형식만 설명 |
| 주민번호 | 매우 높음 | 절대 입력 금지, 형식만 설명 |
| 급여 정보 | 높음 | 임의 금액으로 대체 |
| 고객 정보 | 높음 | 더미 데이터로 대체 |
| 매출 데이터 | 중간 | 비율만 유지한 가상 데이터 사용 |

> 중요: ChatGPT에 입력한 데이터는 서비스 개선에 사용될 수 있습니다. 민감한 업무 데이터는 반드시 더미 데이터로 대체한 후 요청하세요. ChatGPT 설정에서 "대화 기록 학습 끄기" 옵션을 활성화하는 것도 방법입니다.

### 수식 검증하기

ChatGPT가 만들어준 수식을 바로 대량 적용하지 마세요. 다음 단계를 따르세요.

1. **소규모 테스트**: 먼저 5~10행에만 적용해서 결과를 확인합니다.
2. **수동 검산**: 1~2개 행은 계산기로 직접 검산합니다.
3. **엣지 케이스 확인**: 빈 셀, 0, 음수, 매우 큰 수 등 예외적인 데이터에서도 올바르게 작동하는지 확인합니다.
4. **전체 적용**: 검증 완료 후 전체 범위에 적용합니다.

```
ChatGPT에게 추가로 요청하기 좋은 질문:
- "이 수식에서 빈 셀이 있으면 어떻게 되나?"
- "데이터가 10만 행이면 성능 문제가 있을까?"
- "이 수식이 에러 없이 작동하는지 모든 경우를 확인해줘"
```

## 실전 시나리오: 월간 매출 보고서 수식 한 번에 만들기

실제 업무 상황을 가정하고 ChatGPT에게 한 번에 필요한 수식을 모두 요청하는 방법입니다.

### 프롬프트 예시

```
월간 매출 보고서를 만들고 있어. 한 번에 필요한 수식을 모두 만들어줘.

데이터 구조 (Sheet1 "원본데이터"):
- A열: 날짜 (yyyy-mm-dd)
- B열: 영업사원명
- C열: 고객사명
- D열: 제품명
- E열: 수량
- F열: 단가
- G열: 할인율(%)

Sheet2 "월간보고"에 다음 수식이 필요해:
1. G열에 매출액 수식 (수량 x 단가 x (1-할인율))
2. 2026년 3월 전체 매출 합계
3. 영업사원별 매출 합계 (H열에 이름, I열에 합계)
4. 전월 대비 성장률 (2월 매출 대비)
5. 상위 3개 제품 매출 순위
6. 목표 달성률 (목표 금액은 Sheet3의 B2에 있음)
```

### ChatGPT가 만들어주는 결과

ChatGPT는 위 요청에 대해 6개의 수식을 한 번에 제공하며, 각 수식의 입력 위치와 설명도 함께 알려줍니다. 이렇게 한 번에 여러 수식을 요청하면 시간을 크게 절약할 수 있습니다.

## 마무리

엑셀 수식은 더 이상 외울 필요가 없습니다. ChatGPT에게 "이런 결과를 원해"라고 자연어로 설명하면 됩니다. 핵심은 데이터 구조를 명확히 설명하고, 원하는 결과를 구체적으로 요청하는 것입니다.

처음에는 간단한 VLOOKUP이나 IF 수식부터 시작하세요. 익숙해지면 INDEX-MATCH, VBA 매크로까지 ChatGPT의 도움으로 엑셀 실력이 빠르게 향상될 것입니다. 오늘 당장 ChatGPT를 열고 그동안 해결 못 했던 엑셀 문제를 물어보세요.
