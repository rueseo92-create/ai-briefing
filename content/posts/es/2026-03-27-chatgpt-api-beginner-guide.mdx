## Ejemplo práctico de uso de la API de ChatGPT: Crear tu propio chatbot de IA

### Implementación de chatbot básico

Ahora creemos un chatbot como un caso de uso más práctico de la API de ChatGPT. Es una versión que guarda el historial de conversación y permite conversaciones continuas:

```python
class SimpleChatBot:
    def __init__(self, api_key):
        self.client = OpenAI(api_key=api_key)
        self.conversation = []
    
    def chat(self, user_message):
        # Agregar mensaje del usuario
        self.conversation.append({"role": "user", "content": user_message})
        
        # Llamada API
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=self.conversation,
            temperature=0.7
        )
        
        # Guardar respuesta de IA
        ai_message = response.choices[0].message.content
        self.conversation.append({"role": "assistant", "content": ai_message})
        
        return ai_message

# Ejemplo de uso
bot = SimpleChatBot('your-api-key-here')
print(bot.chat("Cuéntame cómo estudiar Python"))
print(bot.chat("¿Podrías explicarlo más específicamente?"))  # Mantiene el contexto de la conversación anterior
```

### Manejo de errores y situaciones excepcionales en el uso de la API de ChatGPT

Este tipo de manejo de errores es esencial al usar la API de ChatGPT en servicios reales:

```python
import time
from openai import RateLimitError, APIError

def safe_api_call(client, messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages
            )
            return response.choices[0].message.content
            
        except RateLimitError:
            if attempt < max_retries - 1:
                time.sleep(2 ** attempt)  # Backoff exponencial
                continue
            else:
                return "Lo siento. El servidor está ocupado actualmente. Por favor, inténtelo de nuevo más tarde."
                
        except APIError as e:
            return f"Ha ocurrido un error: {str(e)}"
```

    stream = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=messages,
        stream=True

    for chunk in stream:

## Preguntas frecuentes (FAQ)

### P1: ¿Cuánto costará usar la API de ChatGPT?

R: En el uso de la API de ChatGPT, basado en los estándares de 2026, al usar el modelo GPT-3.5 Turbo, cuesta $0.001 por entrada y $0.002 por salida por 1000 tokens. Aproximadamente 10 conversaciones típicas costarían menos de $0.1. Si usas la API de ChatGPT en proyectos personales, $5-10 al mes debería ser suficiente.

### P2: ¿Qué debo hacer si mi clave API se filtra durante el uso de la API de ChatGPT?

R: En caso de un incidente de seguridad en el uso de la API de ChatGPT, desactiva inmediatamente esa clave en la plataforma OpenAI y obtén una nueva clave. Gestiona las claves con variables de entorno o archivos de configuración, y nunca las incluyas directamente

en el código fuente.

### P3: ¿Cuáles son los límites de tokens al usar la API de ChatGPT?

R: En 2026, los límites de tokens para el uso de la API de ChatGPT varían según el modelo. GPT-3.5 Turbo soporta hasta 16K tokens, mientras que GPT-4 Turbo puede manejar hasta 128K tokens. Un token equivale aproximadamente a 0.75 palabras en inglés o 1-2 caracteres en español. Para conversaciones largas, es recomendable implementar un sistema que resuma o elimine el historial antiguo.

### P4: ¿Cómo puedo acelerar las respuestas al usar la API de ChatGPT?

R: Para optimizar la velocidad en el uso de la API de ChatGPT, considera estos métodos:
- Usa GPT-3.5 Turbo en lugar de GPT-4 para respuestas más rápidas
- Implementa respuestas de streaming para mostrar resultados en tiempo real
- Reduce el parámetro max_tokens si no necesitas respuestas largas
- Usa caché para preguntas frecuentes
- Optimiza tus prompts para ser más concisos y directos

### P5: ¿Es seguro usar la API de ChatGPT para datos sensibles?

R: OpenAI indica que los datos enviados a través de la API no se usan para entrenar modelos, pero para información altamente sensible, considera implementar encriptación adicional antes de enviar los datos. Para uso empresarial, OpenAI también ofrece opciones de mayor seguridad y privacidad.

### P6: ¿Puedo usar la API de ChatGPT en aplicaciones comerciales?

R: Sí, puedes usar la API de ChatGPT en aplicaciones comerciales. Sin embargo, debes revisar los términos de servicio de OpenAI y asegurarte de cumplir con las políticas de uso. Para aplicaciones de alto volumen, considera contactar directamente con OpenAI para obtener precios empresariales.
