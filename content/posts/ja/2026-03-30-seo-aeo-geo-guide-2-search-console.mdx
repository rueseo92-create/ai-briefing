ブログを作ったのにGoogleやNaverで検索しても出てこない場合は？検索エンジンに「私のサイトがここにあります」と直接教えるプロセスが必要です。この記事では、Google Search ConsoleとNaver Search Advisorの登録を段階別にご案内します。

> 核心回答：Google Search ConsoleとNaver Search Advisorにサイトを登録してsitemap.xmlを提出すると、検索エンジンがブログを認識してクロールを開始します。登録後インデックスまで通常2-7日かかります。

---

## Google Search Console登録方法

Google Search Console（GSC）は、Google検索露出の必須第一歩です。2026年基準で全世界検索市場におけるGoogleシェアは約91%に達します（出典：StatCounter, 2026）。

### GSC登録7段階

1. search.google.com/search-console にアクセス後、Googleアカウントでログイン
2. 「プロパティを追加」をクリック → 「URLプレフィックス」を選択 → `https://seroai.xyz` を入力
3. 所有権の確認 → 「HTMLタグ」方式を選択（最も簡単）
4. メタタグコードをコピーしてNext.jsの `app/layout.tsx` に追加

> コツ：Next.jsでは `metadata` オブジェクトに `verification` フィールドを追加するのが最もスマートな方法です。

コード例：

```typescript
// app/layout.tsx
export const metadata = {
  verification: {
    google: 'GSCで受け取った認証コード',
  },
};
```

5. Vercelにデプロイ → GSCで「確認」をクリック
6. 左側メニュー「サイトマップ」→ `sitemap.xml` を提出
7. `feed.xml` も追加提出（RSSフィード）

### GSCで確認できること

| 項目 | 説明 |
|------|------|
| 検索パフォーマンス | クリック数、インプレッション数、CTR、平均順位 |
| インデックス作成 | Googleが認識したページ数 |
| サイトマップ | 提出したURL処理状態 |
| エクスペリエンス | Core Web Vitalsスコア |

> 要点：GSC登録後「URL検査」ツールで新しい記事を公開するたびにインデックスをリクエストすると、Google露出が2-3日早くなります。

## Naver Search Advisor登録方法

韓国市場をターゲットにするなら、Naver登録は必須です。2026年基準で韓国検索市場におけるNaverシェアは約28%、Googleは約62%です（出典：InternetTrend, 2026）。Naverからの流入も無視できない数値ですね。

### Naver Search Advisor登録8段階

1. searchadvisor.naver.com にアクセス → Naverログイン
2. 「サイト登録」→ `https://seroai.xyz` を入力
3. 所有確認 → 「HTMLタグ」方式を選択
4. メタタグをコピーしてlayout.tsxに追加

```typescript
// app/layout.tsx（Google + Naver同時）
export const metadata = {
  verification: {
    google: 'GSC認証コード',
    other: {
      'naver-site-verification': 'Naver認証コード',
    },
  },
};
```

5. デプロイ後、Naverで「確認」をクリック
6. リクエスト → サイトマップ提出 → `sitemap.xml`
7. リクエスト → RSS提出 → `feed.xml`
8. リクエスト → ウェブページ収集 → メインURL収集リクエスト

> 核心回答：Naver Search Advisorでは、サイトマップ、RSS、ウェブページ収集をすべて提出する必要があります。この3つをすべて提出すると、Naver露出まで平均3-5日かかります。

## 登録後の確認事項

両サービスともに登録したら、次の事項をチェックしてください：

1. サイトマップ状態 — GSCとNaver両方で「成功」表示を確認
2. インデックス状態 — 登録後3-7日後に `site:seroai.xyz` でGoogle検索してインデックス確認
3. エラーチェック — GSC「ページ」タブでクロールエラーがないか確認

> コツ：新しい記事を公開するたびにGSCのURL検査ツールとNaverのウェブページ収集リクエストを使用すると、インデックス速度を大幅に短縮できます。

## よくある質問

### Q1. Search Console登録後に検索に出てこないのですが？
A1. インデックスまで2-7日必要です。サイトマップが正常に提出されたか確認し、URL検査ツールで個別ページのインデックスをリクエストしてみてください。robots.txtでクロールがブロックされていないかも確認してください。

### Q2. URLプレフィックスとドメイン方式のどちらを選べばいいですか？
A2. 個人ブログはURLプレフィックス方式が簡単です。ドメイン方式はDNS設定が必要で複雑ですし、URLプレフィックスですべての機能を使用できます。

### Q3. サイトマップはどのくらいの頻度で更新すべきですか？
A3. Next.jsの自動生成サイトマップを使用すると、毎ビルドごとに自動更新されます。別途再提出する必要はなく、検索エンジンが定期的に確認します。
