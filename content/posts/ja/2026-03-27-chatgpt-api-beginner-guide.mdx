## 実践ChatGPT API活用例：自分だけのAIチャットボット作成

### 基本チャットボット実装

今度は少しより実用的なChatGPT API活用事例としてチャットボットを作ってみましょう。会話履歴を保存し、連続会話が可能なバージョンです：

```python
class SimpleChatBot:
    def __init__(self, api_key):
        self.client = OpenAI(api_key=api_key)
        self.conversation = []
    
    def chat(self, user_message):
        # ユーザーメッセージ追加
        self.conversation.append({"role": "user", "content": user_message})
        
        # API呼び出し
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=self.conversation,
            temperature=0.7
        )
        
        # AI応答保存
        ai_message = response.choices[0].message.content
        self.conversation.append({"role": "assistant", "content": ai_message})
        
        return ai_message

# 使用例
bot = SimpleChatBot('your-api-key-here')
print(bot.chat("Pythonの学習方法を教えて"))
print(bot.chat("もっと具体的に説明してもらえる？"))  # 前の会話のコンテキスト維持
```

### ChatGPT API活用でのエラー処理と例外状況

実際のサービスでChatGPT APIを活用する際は、このようなエラー処理が必須です：

```python
import time
from openai import RateLimitError, APIError

def safe_api_call(client, messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages
            )
            return response.choices[0].message.content
            
        except RateLimitError:
            if attempt < max_retries - 1:
                time.sleep(2 ** attempt)  # 指数バックオフ
                continue
            else:
                return "申し訳ございません。現在サーバーが混雑しています。しばらく後で再度お試しください。"
                
        except APIError as e:
            return f"エラーが発生しました：{str(e)}"
```
