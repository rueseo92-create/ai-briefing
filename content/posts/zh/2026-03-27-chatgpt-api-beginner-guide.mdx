## 实战ChatGPT API应用示例：制作专属AI聊天机器人

### 基础聊天机器人实现

现在让我们用稍微实用一些的ChatGPT API应用案例来制作聊天机器人。这是一个能保存对话记录、支持连续对话的版本：

```python
class SimpleChatBot:
    def __init__(self, api_key):
        self.client = OpenAI(api_key=api_key)
        self.conversation = []
    
    def chat(self, user_message):
        # 添加用户消息
        self.conversation.append({"role": "user", "content": user_message})
        
        # API调用
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=self.conversation,
            temperature=0.7
        )
        
        # 保存AI回复
        ai_message = response.choices[0].message.content
        self.conversation.append({"role": "assistant", "content": ai_message})
        
        return ai_message

# 使用示例
bot = SimpleChatBot('your-api-key-here')
print(bot.chat("告诉我学习Python的方法"))
print(bot.chat("能再具体说明一下吗？"))  # 保持之前对话的上下文
```

### ChatGPT API应用中的错误处理和异常情况

在实际服务中应用ChatGPT API时，这样的错误处理是必需的：

```python
import time
from openai import RateLimitError, APIError

def safe_api_call(client, messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages
            )
            return response.choices[0].message.content
            
        except RateLimitError:
            if attempt < max_retries - 1:
                time.sleep(2 ** attempt)  # 指数退避
                continue
            else:
                return "抱歉。目前服务器繁忙。请稍后再试。"
                
        except APIError as e:
            return f"发生错误：{str(e)}"
```

def stream_chat_response(client, messages):
    stream = client.chat.completions.create(
        model="gpt-3.5-turbo",
        messages=messages,
        stream=True
    )
    
    for chunk in stream:
        if chunk.choices[0].delta.content is not None:
            print(chunk.choices[0].delta.content, end="")
