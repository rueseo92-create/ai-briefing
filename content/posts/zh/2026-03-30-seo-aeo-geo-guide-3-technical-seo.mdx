注册完Search Console后，现在需要创建一个让搜索引擎和AI爬虫能够正确读取网站的环境。这篇文章将介绍技术SEO的4个核心文件。

> 核心答案：技术SEO文件有4种：robots.txt（爬虫规则）、sitemap.xml（页面列表）、feed.xml（RSS订阅）、llms.txt（AI爬虫摘要）。在Next.js中都可以自动或半自动生成。

---

## robots.txt — 爬虫允许设置

robots.txt是告诉搜索引擎爬虫"哪里可以读取，哪里不可以"的文件。在2026年，也需要考虑AI爬虫（GPTBot、Claude-Web等）。

### 推荐的robots.txt设置

```
User-agent: *
Allow: /

# 也允许AI爬虫
User-agent: GPTBot
Allow: /
User-agent: ChatGPT-User
Allow: /
User-agent: Claude-Web
Allow: /
User-agent: PerplexityBot
Allow: /

Sitemap: https://seroai.xyz/sitemap.xml
```

> 小贴士：如果屏蔽AI爬虫，ChatGPT、Perplexity等服务就不会引用博客内容。为了GEO，务必要允许访问。

### 主要AI爬虫列表

| 爬虫 | 服务 | 推荐设置 |
|--------|--------|-----------|
| GPTBot | ChatGPT/OpenAI | Allow |
| ChatGPT-User | ChatGPT 浏览 | Allow |
| Claude-Web | Claude/Anthropic | Allow |
| PerplexityBot | Perplexity AI | Allow |
| Googlebot | 谷歌搜索 | Allow |
| Yeti | Naver搜索 | Allow |

## sitemap.xml — 提供所有页面列表

网站地图是告诉搜索引擎博客所有页面URL和更新日期的文件。根据谷歌官方文档，提交网站地图的网站索引速度平均快4倍（来源：Google Search Central，2026）。

### 在Next.js中自动生成

```typescript
// app/sitemap.ts
export default async function sitemap() {
  const posts = getAllPosts(); // 获取所有文章
  return [
    { url: 'https://seroai.xyz', lastModified: new Date() },
    { url: 'https://seroai.xyz/posts', lastModified: new Date() },
    ...posts.map(post => ({
      url: `https://seroai.xyz/posts/${post.slug}`,
      lastModified: new Date(post.date),
    })),
  ];
}
```

> 核心：在Next.js App Router中创建`app/sitemap.ts`，构建时会自动生成sitemap.xml。发布新文章并部署后会自动更新，无需单独管理。

## feed.xml (RSS) — 订阅和Naver提交用

RSS订阅是博客文章的标准订阅格式。也是Naver Search Advisor的必须提交项目。提交RSS后，Naver能够更快识别新文章。

### RSS生成方法

```bash
npm install feed
```

> 核心答案：feed.xml是必须向Naver Search Advisor提交的文件。使用`feed`包自动生成，在Next.js Route Handler（`app/feed.xml/route.ts`）中提供即可。

---

## llms.txt — 为AI爬虫准备的网站摘要（GEO核心）

llms.txt是2025年提出的新标准，是向AI爬虫提供网站核心信息摘要的文件。如果说robots.txt是"读这里"，那么llms.txt就是"我们网站是这样的"的说明作用。

### 推荐的llms.txt结构

```markdown
# 网站名称
> 一句话介绍

## 主要内容
- 分类1：说明
- 分类2：说明

## 网站信息
- URL: https://seroai.xyz
- 语言：中文
- 更新频率：每日
- RSS: /feed.xml
- 网站地图: /sitemap.xml
```

> 核心：llms.txt部署到`public/llms.txt`即可。AI爬虫读取此文件后会更好地理解网站，提高在AI答案中被引用的概率。

## 常见问题

### Q1. 在robots.txt中屏蔽AI爬虫会怎样？
A1. ChatGPT、Perplexity等AI服务不会引用博客内容。会失去流量引入机会，如果想要GEO效果，必须允许访问。

### Q2. llms.txt一定要制作吗？
A2. 不是必须的，但强烈推荐。AI爬虫能更好地理解网站结构，在相关问题回答中被引用的可能性会提高。制作只需要5分钟。

### Q3. sitemap.xml需要包含所有页面吗？
A3. 只需要包含想要在搜索中展示的页面即可。排除404页面、管理员页面等。使用Next.js自动生成的话，只会自动包含`app`目录的公开页面。
