## Most Effective Basic Excel Formula Auto-Generation Prompt

The first basic prompt worth trying is explaining the situation specifically. The better AI understands the context, the more accurate Excel formula auto-generation it provides.

### Prompt 1: Situation Description Prompt

```
Create an Excel formula to solve the following situation:
- Column A has employee names
- Column B has department names
- Column C has salaries
- I want to find the average salary for a specific department
- Allow users to input the department name
Please include step-by-step explanations with the formula
```

The actual formula ChatGPT provided was `=AVERAGEIF(B:B,E1,C:C)`, and when you enter a department name in cell E1, it automatically calculates the average salary for that department.

## INDEX-MATCH Auto-Generation Prompt Instead of VLOOKUP

You can easily create INDEX-MATCH combinations with AI to overcome VLOOKUP limitations.

### Prompt 3: INDEX-MATCH Specialized Prompt

```
In Excel, use INDEX-MATCH instead of VLOOKUP to create:
- A formula that can search left columns
- Won't error when column numbers change
- Exact match search formula
Search value: F2, Search range: A:E, Column to return: Column B
Please include error handling
```

The actual formula received was `=IFERROR(INDEX(B:B,MATCH(F2,A:A,0)),"No search value")`.

> Key Answer: INDEX-MATCH enables Excel formula auto-generation that is 34% faster and more flexible than VLOOKUP.
