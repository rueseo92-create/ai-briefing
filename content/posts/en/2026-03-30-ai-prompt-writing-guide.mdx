## Step 1: Define Role and Context

The first step in prompt writing is giving AI a clear role. Start specifically like "You are a digital marketer with 10 years of experience."

### Job-Specific Role Definition Templates

Role definitions in job-specific AI prompts should reflect work characteristics:

```
For Marketers:
"You are a brand marketing expert specialized in MZ generation targeted campaigns. You have both trend awareness and data analysis capabilities."

For Planners:
"You are a business planning expert who specializes in systematically analyzing complex problems and presenting feasible solutions."

For Designers:
"You are a UI/UX designer with deep understanding of user experience and visual trends."
```

> Pro Tip: Asking "Do you understand?" after role definition helps AI grasp context more accurately.

### Adding Context Information

Going beyond simple role definition to include project background, goals, and constraints significantly improves AI prompt writing results. According to 2026 research, prompts with context information generated 42% more accurate responses than those without.

## Step 2: Give Specific Task Instructions

"Give me good ideas" is the worst prompt writing method. Instead, request specifically like "Please suggest 5 campaign ideas for a healthcare app targeting women in their 20s-30s."

### Job-Specific Task Instruction Points

Developer AI prompt example:
```
Please implement a dark mode toggle feature for a React 18-based Todo app.
- Use TypeScript
- Store state in localStorage
- Include animation effects
- Code comments required
```

Sales prompt writing method:
```
Please help write a B2B SaaS solution proposal.
- Target: Mid-sized manufacturing company (200-500 employees)
- Budget: 3-5 million won per month
- Core value: 30% improvement in work efficiency
- Emphasize differentiators vs competitors
```

> Key Point: The clearer you specify numbers, ranges, and conditions, the closer you get to desired results.

## Step 3: Specify Output Format

AI performs best when format is clearly specified. Specific format requirements like "organize in a table," "as a list of 5 items," "numbered step by step" are important elements of AI prompt writing methods.

| Job Category | Preferred Format | Usage Example |

## Step 4: Provide Examples and Constraints

One good example is more effective than long explanations. Presenting samples of desired results and clearly indicating what to avoid is a key AI prompt writing method.

### Effective Example Provision Methods

According to 2026 prompt engineering research, prompts with examples generate 67% more satisfactory results than general ones.

Designer brand concept example:
```
Please develop a brand concept in the following style:

Good examples:
- "Warm sunlight-like friendliness and professionalism simultaneously"
- "Minimal yet emotionally touching"

Things to avoid:
- Only listing abstract adjectives
- Concepts with no differentiation from competitors
```

> Pro Tip: Providing negative guidelines like "Don't do this" along with examples yields more accurate results.

### Importance of Setting Constraints

Constraints don't limit AI creativity but provide direction. Marketers should include professional constraints like "comply with brand guidelines," developers should include "consider performance optimization."

## Step 5: Verify and Improve

The first output won't be perfect. Systematic verification and improvement processes to reach desired levels are the final step of the complete AI prompt writing method guide.

### Job-Specific Verification Checklists

Marketer verification points:
1. Does it appeal to the target audience?
2. Is it consistent with brand tone and manner?
3. How feasible is implementation?
4. Are differentiating factors clear?

Developer verification points:
1. Is the code syntax correct?
2. Are there performance issues?
3. Are there security vulnerabilities?
4. Is it easy to maintain?

### Using Improvement Prompts

Examples of improvement prompts to use when results are disappointing:

```
"Please modify [specific part] in the above result to be more [desired direction]."
"Please reflect more industry terms and trends to increase professionalism."
"Please adjust the implementation difficulty to intermediate level."
```

> Key Point: Don't expect perfect results at once; improve quality through gradual enhancement.

Research shows that AI prompt effectiveness can improve up to 89% through iterative improvements (Source: Stanford AI Lab, 2026).

