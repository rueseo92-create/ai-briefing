## Practical ChatGPT API Example: Creating Your Own AI Chatbot

### Basic Chatbot Implementation

Now let's create a chatbot as a more practical ChatGPT API use case. This version saves conversation history and enables continuous dialogue:

```python
class SimpleChatBot:
    def __init__(self, api_key):
        self.client = OpenAI(api_key=api_key)
        self.conversation = []
    
    def chat(self, user_message):
        # Add user message
        self.conversation.append({"role": "user", "content": user_message})
        
        # API call
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=self.conversation,
            temperature=0.7
        )
        
        # Save AI response
        ai_message = response.choices[0].message.content
        self.conversation.append({"role": "assistant", "content": ai_message})
        
        return ai_message

# Usage example
bot = SimpleChatBot('your-api-key-here')
print(bot.chat("Tell me how to study Python"))
print(bot.chat("Can you explain in more detail?"))  # Maintains previous conversation context
```

### Error Handling and Exception Cases in ChatGPT API Usage

Error handling like this is essential when using ChatGPT API in actual services:

```python
import time
from openai import RateLimitError, APIError

def safe_api_call(client, messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            response = client.chat.completions.create(
                model="gpt-3.5-turbo",
                messages=messages
            )
            return response.choices[0].message.content
            
        except RateLimitError:
            if attempt < max_retries - 1:
                time.sleep(2 ** attempt)  # Exponential backoff
                continue
            else:
                return "Sorry, the server is currently busy. Please try again in a moment."
                
        except APIError as e:
            return f"An error occurred: {str(e)}"
```
