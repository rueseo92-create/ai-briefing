블로그를 만들었는데 구글이나 네이버에서 검색해도 안 나온다면? 검색엔진에 "내 사이트가 여기 있어요"라고 **직접 알려주는 과정**이 필요합니다. 이 글에서는 구글 서치 콘솔과 네이버 서치 어드바이저 등록을 단계별로 안내합니다.

> **핵심 답변:** 구글 서치 콘솔과 네이버 서치 어드바이저에 사이트를 등록하고 sitemap.xml을 제출하면, 검색엔진이 블로그를 인식하고 크롤링을 시작합니다. 등록 후 색인까지 보통 **2~7일**이 소요됩니다.

## Google Search Console 등록 방법

구글 서치 콘솔(GSC)은 구글 검색 노출의 **필수 첫 단계**입니다. 2026년 기준 전 세계 검색 시장에서 구글 점유율은 **약 91%**에 달합니다 (출처: StatCounter, 2026).

### GSC 등록 7단계

1. **search.google.com/search-console** 접속 후 Google 계정 로그인
2. **"속성 추가"** 클릭 → **"URL 접두사"** 선택 → `https://seroai.xyz` 입력
3. **소유권 확인** → **"HTML 태그"** 방식 선택 (가장 간편)
4. 메타태그 코드를 복사하여 Next.js `app/layout.tsx`에 추가

> **꿀팁:** Next.js에서는 `metadata` 객체에 `verification` 필드를 추가하는 것이 가장 깔끔한 방법입니다.

코드 예시:

```typescript
// app/layout.tsx
export const metadata = {
  verification: {
    google: 'GSC에서_받은_인증코드',
  },
};
```

5. Vercel에 배포 → GSC에서 **'확인'** 클릭
6. 좌측 메뉴 **"사이트맵"** → `sitemap.xml` 제출
7. `feed.xml`도 추가 제출 (RSS 피드)

### GSC에서 확인할 수 있는 것들

| 항목 | 설명 |
|------|------|
| 검색 실적 | 클릭수, 노출수, CTR, 평균 순위 |
| 색인 생성 | 구글이 인식한 페이지 수 |
| 사이트맵 | 제출한 URL 처리 상태 |
| 경험 | Core Web Vitals 점수 |

> **핵심:** GSC 등록 후 **"URL 검사"** 도구로 새 글을 발행할 때마다 색인을 요청하면, 구글 노출이 **2~3일 더 빨라**집니다.

## 네이버 서치 어드바이저 등록 방법

한국 시장을 타깃한다면 네이버 등록은 필수입니다. 2026년 기준 한국 검색 시장에서 네이버 점유율은 **약 28%**, 구글은 **약 62%**입니다 (출처: InternetTrend, 2026). 네이버 유입도 무시할 수 없는 수치죠.

### 네이버 서치 어드바이저 등록 8단계

1. **searchadvisor.naver.com** 접속 → 네이버 로그인
2. **"사이트 등록"** → `https://seroai.xyz` 입력
3. **소유 확인** → **"HTML 태그"** 방식 선택
4. 메타태그를 복사하여 layout.tsx에 추가

```typescript
// app/layout.tsx (구글 + 네이버 동시)
export const metadata = {
  verification: {
    google: 'GSC_인증코드',
    other: {
      'naver-site-verification': '네이버_인증코드',
    },
  },
};
```

5. 배포 후 네이버에서 **'확인'** 클릭
6. **요청 → 사이트맵 제출** → `sitemap.xml`
7. **요청 → RSS 제출** → `feed.xml`
8. **요청 → 웹 페이지 수집** → 메인 URL 수집 요청

> **핵심 답변:** 네이버 서치 어드바이저에서는 사이트맵, RSS, 웹 페이지 수집을 모두 제출해야 합니다. 세 가지를 모두 제출하면 네이버 노출까지 평균 **3~5일**이 걸립니다.

## 등록 후 확인사항

두 서비스 모두 등록했다면, 다음 사항을 점검하세요:

1. **사이트맵 상태** — GSC와 네이버 모두에서 "성공" 표시 확인
2. **색인 상태** — 등록 후 3~7일 뒤 `site:seroai.xyz`로 구글 검색하여 색인 확인
3. **오류 체크** — GSC "페이지" 탭에서 크롤링 오류가 없는지 확인

> **꿀팁:** 새 글을 발행할 때마다 GSC의 URL 검사 도구와 네이버의 웹 페이지 수집 요청을 사용하면 색인 속도를 크게 단축할 수 있습니다.

## 자주 묻는 질문

### Q1. 서치 콘솔 등록 후 검색에 안 나오는데요?
A1. 색인까지 **2~7일**이 필요합니다. 사이트맵이 정상 제출되었는지 확인하고, URL 검사 도구로 개별 페이지 색인을 요청해보세요. robots.txt에서 크롤링이 차단되어 있지 않은지도 확인하세요.

### Q2. URL 접두사와 도메인 방식 중 어떤 걸 선택하나요?
A2. 개인 블로그는 **URL 접두사** 방식이 간편합니다. 도메인 방식은 DNS 설정이 필요해서 복잡하고, URL 접두사로도 모든 기능을 사용할 수 있습니다.

### Q3. 사이트맵은 얼마나 자주 업데이트해야 하나요?
A3. Next.js의 자동 생성 사이트맵을 사용하면 매 빌드마다 자동 업데이트됩니다. 별도로 재제출할 필요 없이, 검색엔진이 주기적으로 확인합니다.
